# Design DNA — Satúrnia

## Visual Personality: dark-modern (observatorio de lujo cósmico)
## Layout: storytelling — inmersión total, sección por sección como avanzar por el sistema solar
## Typography: elegant-serif — DM Serif Display + DM Sans

### Color Palette
- Primary Brand (Gold): #C9A84C — oklch(0.72 0.10 82)
- Surface Base (Space Black): #0A0A14 — oklch(0.08 0.015 265)
- Surface Mid (Midnight Blue): #12122A — oklch(0.11 0.025 275)
- Text Primary: oklch(0.95 0.005 80) — warm cream white
- Text Muted: oklch(0.65 0.015 80) — dusty gold-gray

### Font Pairing
- Display: DM Serif Display — weight 400 (astronomía vintage + editorial de moda)
- Body: DM Sans — weight 300–500 (limpio, moderno, legible sobre oscuro)

### Unique Design Choices
- Saturn ring SVG accent dividers — anillos dorados que separan el cosmos entre secciones
- Noise grain overlay en el hero — textura de terciopelo cósmico (feTurbulence inline SVG)
- Dot star-field pattern via CSS radial-gradient masked con radial fade
- Gold metallic shimmer en headlines — CSS gradient text via background-clip
- Asimetría intencional: secciones alternadas left-heavy y centradas para narrativa de avance
- Cada sección es una zona distinta del sistema solar — diferentes profundidades de oscuridad
